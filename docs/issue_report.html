AdSDK Issue Report

Thank you for your contribution to improving the AdSDK.

Since we use private keys and identifiers when working with AdSDK, we accept private issue reports by email.

To create an issue report, please:

  • Prepare any additional data you can provide us with: logs, stack traces, etc. Please note that you don't have debug symbols if you use Swift Package Manager to integrate SDK.
    If you need debug symbols to better understand the issue, you can request an SDK with them from us.
  • Prepare a demo app where we can reproduce the issue. You can use our template as a starter project and edit it to reproduce the error.
  • Follow the link and fill in as much information as possible. In addition, don't forget to attach the demo app and the files described in the steps above.
